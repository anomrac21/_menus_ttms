üí° Implementation Tips

  • Mobile First: All buttons are responsive and work great on mobile devices.
  • Performance: Icons are lazy-loaded for optimal performance.
  • Accessibility: All buttons include ARIA labels and are keyboard-navigable.
  • Customization: Override the CSS classes to match your brand colors.
  • Instagram & TikTok: These use Web Share API on mobile, clipboard on desktop.
  • Profile Links: Add username parameters to link directly to your social profiles.
